# Portfolio de Job Yauri

Portafolio web moderno y responsivo para desarrollador full stack.

## Características implementadas

### ✅ Animaciones al hacer scroll
- Elementos aparecen suavemente al entrar en pantalla
- Secciones se deslizan o desvanecen con Intersection Observer

### ✅ Tarjetas de proyectos dinámicas
- Cada proyecto muestra imagen, título, descripción, tecnologías
- Botones "Ver Demo" y "Ver Código" con overlay hover
- Efectos hover que levantan las tarjetas

### ✅ Iconos grandes de tecnologías
- Sección de habilidades con iconos grandes para HTML, CSS, JavaScript, PHP, MySQL
- Animaciones hover en los iconos

### ✅ Estadísticas animadas
- Contadores que se animan automáticamente al aparecer:
  - +10 proyectos realizados
  - +2 años aprendiendo programación
  - +5 tecnologías dominadas

### ✅ Sección de experiencia/educación
- Tarjetas visuales con iconos para:
  - 🎓 Ingeniería de Software
  - 💻 Proyectos académicos
  - 🚀 Desarrollo Full Stack

### ✅ Botón descargar CV
- Botón destacado que descarga archivo PDF del CV

### ✅ Sección de contacto mejorada
- Enlaces con iconos para GitHub, LinkedIn, WhatsApp, Email

### ✅ Fondo moderno animado
- Gradientes animados en el hero
- Partículas flotantes usando particles.js
- Efectos visuales modernos

### ✅ Diseño general
- Estilo moderno tipo portafolio de desarrollador
- Colores oscuros con acentos azules/celestes/dorados
- Tipografía moderna (Syne + DM Sans)
- Diseño totalmente responsive
- Navegación con menú fijo

## Secciones del sitio
1. Home - Hero con animaciones
2. About - Información personal
3. Skills - Tecnologías con iconos grandes
4. Projects - Tarjetas con hover effects
5. Experience - Educación y experiencia
6. Contact - Formulario y enlaces sociales

## Tecnologías utilizadas
- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6+, Intersection Observer)
- Particles.js para efectos de fondo
- Font Awesome para iconos

## Cómo ejecutar
1. Abrir `index.html` en un navegador web
2. O usar un servidor local: `python -m http.server 8000`

## Archivos
- `index.html` - Estructura principal
- `css/styles.css` - Estilos y animaciones
- `js/script.js` - Funcionalidad JavaScript
- `cv.html` - Versión HTML del CV (guardar como PDF)
- `img/` - Imágenes del portafolio
